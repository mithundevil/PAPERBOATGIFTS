# Paperboat Gifts - Luxury Bespoke E-commerce

A high-end, premium e-commerce platform for curated personalized gifts.

## Brand Experience

- **Luxury Minimal UI**: Expansive, high-contrast design inspired by Apple, Nike, and Dior.
- **Cinematic Homepage**: Full-screen immersive hero with floating product visuals and smooth Framer Motion transitions.
- **Bespoke Product Pages**: Spacious split layout focusing on material quality and artisanal craft.
- **Registry Flow**: A refined, minimalist cart and acquisition sequence for a premium shopping experience.
- **Mobile Optimized**: A fully responsive, high-fidelity experience across all devices with custom typography.

## Technical Excellence

- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS v4, Framer Motion for cinematic animations.
- **Backend**: Node.js, Express, MongoDB (Mongoose), Cloudinary for high-res image hosting.
- **Acquisition**: Razorpay integration with secure server-side signature verification.

## Getting Started

### Installation

1. **Backend**:
   ```bash
   cd backend
   npm install
   npm start
   ```
2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
